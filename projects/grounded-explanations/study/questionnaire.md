# Questionnaire Instruments

Rate each statement **1 = Strongly disagree** … **7 = Strongly agree**.

Administer **after each condition** (Baseline and Modified separately).

---

## Post-task survey (per condition)

### Primary measures

1. **Transparency:** I understood why the system suggested this movie.
2. **Trust:** I would rely on this recommendation to start my search.
3. **Control / steerability:** I felt I could influence the recommendation with my answers.

### Secondary measures

4. **Evidence quality:** The explanation seemed grounded in real reviews or my inputs, not generic text.
5. **Cognitive load / ease:** The explanation felt easy to process.

---

## End-of-session comparative questions

Ask once after both conditions:

1. Which version made you **more willing to watch** the suggested title?
   - [ ] Baseline  [ ] Modified  [ ] Same

2. Which version felt **more honest about the limits of the AI**?
   - [ ] Baseline  [ ] Modified  [ ] Same

---

## Pre-session background (once)

1. How many movies do you typically watch per month? __________
2. How often do you use movie recommendation features (Netflix, Letterboxd, etc.)?
   - [ ] Never  [ ] Rarely  [ ] Sometimes  [ ] Often  [ ] Very often

---

## Debrief interview (semi-structured)

1. Where did the explanation **help** or **annoy** you?
2. Did anything feel **made up or unsupported**? If so, which phrase?
3. What **one change** would most improve trust?
4. Was there any **confusion** about which inputs drove the recommendation?

Record notes verbatim where possible; tag with participant ID (P1–P5).
