# Participant Data Privacy

## What is NOT in this repository

Raw or identifiable **user study data** from the Modify & Eval project is **intentionally excluded** from this GitHub repository, including:

- Individual Likert scores linked to participants
- Debrief interview transcripts or verbatim quotes
- Session recordings, screen captures with personal context
- Email, names, or other contact information
- Session logs that could identify a participant

## Why

This project followed a small within-subjects study (N=5) with classmates and friends. Publishing per-participant data in a public repo would risk **re-identification** and is unnecessary for demonstrating the **system modification** and **evaluation design**.

## What IS in the repository

- Runnable prototype code (`rebert/_prototype_7_1_/`)
- Study protocol, questionnaire, and scenario materials
- Aggregated / illustrative results in the **final report** (submitted separately to the course)
- Technical verification logs (`docs/VERIFICATION.md`) — no human-subjects scores
- A **synthetic example** of structured explanation output (`study/results/example_structured_explanation.json`) for UI demonstration only

## For instructors

Full quantitative tables, thematic coding summaries, and interpretation appear in the **course final report**. Contact the author if access to de-identified raw sheets is required for grading.
