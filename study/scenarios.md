# Study Scenarios

Use **one scenario per condition** so participants do not repeat the same decision context.

## Scenario A — Low-energy Sunday

> It's Sunday evening. You are tired and want something easy to watch at home — not too demanding, but still enjoyable. You have about two hours and might pause halfway through.

**Suggested assignment:** Condition A for Group 1 (Baseline first), Condition B for Group 2 (Modified first).

## Scenario B — Date night

> You and a partner are choosing a movie for date night. You want something you can both enjoy — engaging enough to discuss afterward, but not so niche that only one person would like it.

**Suggested assignment:** Second condition for all participants (whichever version they did not use first).

## Scenario C — Solo reflective / cerebral (backup)

> You are watching alone and want something thoughtful — a film that gives you something to reflect on, even if it is slower or more serious.

Use if a participant has already seen both A and B in prior pilots, or for a third pilot session.

## Facilitator script (before each condition)

1. “Please imagine this situation: [read scenario].”
2. “Use Rebert to get a recommendation that fits this situation. Answer the short questions honestly.”
3. “After you see the recommendation, take a moment to read any explanation shown, then ask Rebert one follow-up question about a movie on screen.”
4. “When you're done, I'll ask you a few survey questions before we switch versions.”
